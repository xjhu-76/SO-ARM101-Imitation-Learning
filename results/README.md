# Results Directory

This directory contains training results and visualizations.

## Contents

- `training_curves.png` - Comparison of ACT vs Pi0.5 training loss
- `training_curves.pdf` - Vector version for publications
- `act_training_log.txt` - Raw training logs for ACT
- `pi05_training_log.txt` - Raw training logs for Pi0.5

## Generating Visualizations

```bash
cd scripts
python plot_training_curves.py
```
